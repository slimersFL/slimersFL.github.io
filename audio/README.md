# Audio files

Drop the 12 album MP3s in this folder using these **exact** filenames:

1. `01-im-a-slimer-part-1.mp3`
2. `02-down-low-too-slow.mp3`
3. `03-puppy-love.mp3`
4. `04-dumb-enough.mp3`
5. `05-no-frillz.mp3`
6. `06-crime2slime.mp3`
7. `07-lemmy-is-god.mp3`
8. `08-banana-bread.mp3`
9. `09-right-on-slime.mp3`
10. `10-im-a-slimer-part-2.mp3`
11. `11-dutchmans-key.mp3`
12. `12-partyin-with-slimer.mp3`

The player on the site will auto-discover them by name. If you want different filenames, edit `ALBUM_TRACKS` near the top of `app.js`.

Until real MP3s are dropped in, the player will show a friendly "audio file not found" message for each track — the rest of the site still works fine.
